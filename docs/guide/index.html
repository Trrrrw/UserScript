# 如何安装/使用脚本？

要使用任何脚本，首先需要浏览器安装 Tampermonkey 脚本管理器扩展（Chromeopen in new window / Firefoxopen in new window / Edgeopen in new window）。

  • 前往脚本列表页面，点击表格 [安装] 进入 GreasyFork 网页，点击 [安装脚本] 后弹出扩展提示，再点击 [安装] 即可。

其他基于 Chromium 内核的浏览器（如国内套皮浏览器）一般都可以使用 Chrome 扩展。
请确保使用 Tampermonkey 正式版 扩展，其他的用户脚本管理器可能导致无法正常使用脚本。
如果要重装脚本，请记得在 Tampermonkey 扩展的回收站中彻底删除脚本后再去重新安装脚本。
不会离线安装 .crx 扩展？Chrome、Edge 重新开启隐藏的 [拖入安装 .crx 扩展] 功能！open in new window
